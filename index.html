Visualization Guide

The map has the zooming option. The current population can be found by hovering over any country in the map by tooltip option. The details information about historic population obervable on click which also visualizes country's map. Comparing line charts available by holding the shfit button while clicking on desired countries to compare.

Visualization Options:

Zooming area by double click
Detail information on click
Comparing Line chart
Detail Historical Graph
Country detail with visual Flag

Click countries to view history

Visualization Type

This is a narrative visualization containing scenes, annotations, parameters, and triggers.

Hybrid Structure

The hybrid structure was followed in this presentation with drill down story. As a result the viewer can follow a guided instruction to visualize data as well as interctive map and graph.

Scense

This hybrid interactive visulization shows follwing scenes:

  • Choropleth map with current population of the country higlighting with different color
  • The higlighting country graph shows detail history with country's flag
  • The comparing option in the extended chart give the user the opportuinity to compare countries historical population

Annotations

  • On hovering over over map gives the user the instruction to click for more details view of the country
  • Historical population graph provides the user information about shift+click option to compare population data
  • Below the chart it demonstrates the color of the particular country once selection for comparision is done

Parameters

Several parameters has been used in this visualization

  • The number of current population through map tooltip
  • Historical population graph for detailed infomation of 56 years data and population growth
  • Comparing option of the population growth and current situation
  • The particular country's flag as a symbol

Triggers

Following triggers has been used for this interactive hybrid visulization

  • A button to show the user the chart, interactive map, and description as well as scrolling oprtion
  • Tooltip option to provide viewer current data about specific country with guided option of details
  • The graphical demonstration to undesrtand the copmparing data of particular countries of choice
